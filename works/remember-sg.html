서강해 그리고 기억해

Crowdfunding Website

Description

사진작가 최근우의 졸업 전시 프로젝트, 《서강해 그리고 기억해》의 크라우드 펀딩 사이트입니다. 서강대학교의 다양한 모습을 사진으로 남기는 프로젝트의 의도에 맞추어, 기존 크라우드 펀딩 사이트를 이용하기보다 서강대 학생이 사이트를 만들어주면 좋겠다는 의뢰를 받고 작업하게 되었습니다.

Themeforest의 유료 템플릿을 가져와 디자인을 수정하고 레이아웃을 재배치했으며, Github Pages에 올리고 도메인을 붙였습니다.
