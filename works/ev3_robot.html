EV3 Robot

Maze Solving Robot with BlockChain

Description

Lego Mindstorms EV3를 이용해 소프트웨어공학 수업중 6인 1팀으로 한 학기동안 진행한 프로젝트입니다.

  • Date

    June 23rd, 2017

  • Skill

    Java, Android Studio, Flask

  • Github

    llb1026/EV3-Robot


Details

주어진 미로 판의 검정색 선을 따라 미로를 해결하는 로봇입니다. 만약 나아가는 도중 주어진 경로인 검정색 선을 벗어나면 Color Sensor로 다시 경로를 찾아 돌아옵니다. 전방의 빨간색, 파란색, 노란색의 벽을 만나면 규칙에 따라 좌우로 방향을 전환하고, 검정색 벽을 만나면 프로그램을 종료합니다. 프로그램이 종료되면 시작점부터 종료지점까지의 경로를 핸드폰에 출력하고, 핸드폰에서 경로 정보 전송 버튼을 클릭하면 해당 정보가 단순한 숫자 배열로 표현되어 플라스크 기반의 블록체인에 올라가 하나의 트랜잭션을 형성합니다. 이렇게 트랜잭션 10개가 모이면 블록 하나가 만들어집니다.
